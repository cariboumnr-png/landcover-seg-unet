# Contributing

This project is maintained by the Ontario Ministry of Natural Resources (MNR). External contributions are welcome following the guidelines
below.

## Primary Developer

This repository was originally developed by Wenxi Feng (MNR), who serves as
the principal contributor and maintainer.

For development-related questions, contact: wenxi.feng@ontario.ca

## How to Contribute

1. Fork the repository and create a feature branch.
2. Ensure your changes follow PEP 8 and include descriptive commit messages.
3. Submit a pull request using GitHub.
4. All contributions will be reviewed for:
   - technical quality
   - security considerations
   - alignment with MNRF and OPS policies

## Licensing

By submitting a contribution, you agree that your work will be licensed under
the Apache License 2.0 and may be redistributed under that licence. All
contributions must be compatible with OPS intellectual property requirements.

## Code of Conduct

Contributors are expected to behave respectfully and professionally. Any
inappropriate behaviour may result in rejection of contributions.
